# Units, Box Model & Fonts Assignment

## Project Overview

This project is a simple web page created using HTML and CSS. It demonstrates the use of CSS units, the box model, and basic styling techniques. The page contains an image with a styled button placed below it.

## Features

* Responsive layout using viewport units (`vw` and `vh`)
* Proper use of the CSS Box Model
* Gradient-styled button
* Centered content using auto margins
* Clean and simple design

## Technologies Used

* HTML5
* CSS3

## CSS Concepts Used

* Universal Selector (`*`)
* Box Sizing (`box-sizing: border-box`)
* Viewport Units (`vw`, `vh`)
* Margin and Padding
* Width and Height
* Linear Gradient
* Font Size and Font Weight
* Text Alignment
